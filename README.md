# theRightOne
マッチングアプリのクローンサイトです

# Description
主にTinderの仕様を参考にし、
・登録/退会/ログイン/ログアウト
・プロフィール編集
・ライク/ディスライク機能　
　→　24時間でlikeを送れる数を制限（5つ）、一度判定したユーザーは二度と表示されない
・マッチ機能　
　→　お互いにlikeを送りあえばマッチ
・メッセージ機能
　→ページネーション機能、最新メッセージが常に最上部に表示されるメッセージタブ
・問い合わせ/通報機能
・管理者画面
　→ユーザーの問い合わせを閲覧、ユーザー情報を編集・削除可能
等の機能を作成しました。

# Requirements
XAMPP 7.0.26
